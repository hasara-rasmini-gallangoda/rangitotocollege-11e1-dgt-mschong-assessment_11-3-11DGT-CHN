GENETICS
Structure of DNA

DNA is a double stranded molecule twisted into a double helix.

It is made up of repeating sub units called nucleotides.

Each nucleotide is made up of a phosphate sugar (deoxyribose),and a base joined together.

There are 4 bases that are paired together by complementary base pairing:

Adenine(A) - Thymine(T)

Cytosine(C) - Guanine(G)

Homologous chromosomes

Individuals inherit two copies of each chromosome - one from each parent.

therefore every individual carries two copies of the same gene.However,they may carry different alleles.

sister chromatids are duplicated chromosomes that are connected to one another at their centromere.

They have the same alleles as they are copies of one another.


How is the genetic code read?

The genetic code is read 3 bases at a time on the DNA base sequence,known as a triplet

Each tiplet codes for one amino acid.

The order of the amino acid sequence determines the final protein made

Each amino acid is connected to another by peptide bonds,to form a polypetide chain.

Mutations

A mutation is a random,permanent,change in the DNA base sequence.

They are the only source of new alleles(and therefore genetic variation) in a species.


point mutations are mutations that only affect a single/few bases

there are four types of point mutations:

Substitution

One base is swapped out for another

Insertion

A base is added to the DNA base sequence

Deletion

A base is removed from the Dna sequence

Definitions

DNA: Deoxyribonucleic Acid - A molecule that carries genetic information.

Chromosomes: thread-like structures that contain long lengths of dna.

Gene: A section of DNA that codes for a particular trait/characteristic.

Allele: Alternate forms of a gene.

Mutations: Random,permanent changes in the DNA base sequence.

Homologous Chromosomes: pairs of chromosomes that are the same length,and carry the same genes at the same location (loci) as each other.
